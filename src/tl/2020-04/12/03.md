---
title:  Panahon Para Muling Tumuklas
date:  14/12/2020
---

Nang hilingan si Moises para pangunahan ang mga Israelita palabas ng Ehipto, malinaw na nawala ng karamihan ang kanilang pananaw bilang mga anak ng Diyos. Kailangang muli nilang matuklasan kung sino ang Diyos na humihingi ng kanilang pagsamba at nagbibigay sa kanila ng maraming mga pangako para sa kahanga-hangang hinaharap. Ang Sabbath ang sentro ng kanilang karanasan para matuto sa kanilang paglalakbay para muling makatuklas. Ito rin ang naging malinaw na tanda sa ibang bansa nang espesyal na relasyon ng Diyos at ng bansang ito. Ang karanasan tungkol sa manna ang bumubuo ng paraan ng Diyos para turuan ang mga Israelita.

`Sa Exodo 16:14-29, anong mga liksyon ang nandoon para matuto ang mga Israelita?`

Nagkakaloob ang Diyos ng mga kababalaghan ng manna para sa mga Israelita, na binibigyan lamang sila ng sapat na pagkain para sa bawat araw. Kung bibigyan Niya sila nang higit kaysa sapat, maaari silang makalimot kung sino ang nagbigay sa kanila. Kaya't sa bawat araw na gumagawa Siya ng milagro para sa kanila, at nakita nila kung paano nag-aaruga ang Diyos. Gayunman, sa araw ng Sabbath kakaiba ang nangyari, kung paano ang araw ay dapat na maging espesyal. Ngayon ay dalawang milagro ang naganap: dalawang dami ng pagkain sa araw ng Biyernes at ang pagkain ay hindi nasira sa magdamag. Dito ang Sabbath ay ginamit upang humanga sa Diyos na kanilang tagapagligtas ang mga Israelita at muling tuklasin kung ano ang ibig sabihin ng pagiging bayan ng Diyos.

Ang mga Israelita ay kakain ng manna sa loob ng 40 taon (Exodo 16:35). Inutusan din ng Diyos si Moises na magtago ng isang omer ng manna para paalalahanan ng mga Israelita kung paano Niya pinakain sila sa ilang (Exodo 16:32, 33) Ito rin ay magiging isang paalala para sa partikular na karanasan sa araw ng Sabbath. Mayroon ring ibang okasyon na nilinaw ng Diyos sa mga Israelita na ang Sabbath ay espesyal.

Ang Sabbath ay isang paraan ng Diyos para tulungan ang mga Israelita para matuklasang muli ang kanilang pagkakakilanlan at ng kanilang Diyos. Hinilingan silang sumunod at ingatang banal ang Sabbath, ngunit ito'y ayon sa konteksto para palawakin ang isang mas malalim na pagkaunawa ng karakter ng kanilang Manlilikha at tungkol sa pagtatatag ng walang katapusang relasyon ng pangako.

`Ikaw ay nakikipag-usap sa isang kabataan na nararanasang ang Sabbath ay "nakababagot." Nangingilin lamang siya dahil ito ang sinasabi ng Biblia at ng kanyang mga magulang na dapat gawin. Anong mga mungkahi ang maibibigay mo para matulungan siyang (muling) matuklasan ang Sabbath bilang isang positibong karanasan para matuto?`
